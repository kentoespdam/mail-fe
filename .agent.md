## Stack & Standards
- **Tech:** Next.js 16.2.1 (App Router), React 19, TanStack Query v5, TanStack Table v8, React Hook Form v7, Zod v4.
- **Styling:** Tailwind CSS v4, shadcn/ui v4, Tabler Icons.
- **Linting:** Biome (v2.2.0).
- **Architecture:** Layered separation (Types -> API -> Hooks -> Components -> Page).

## Generation Pattern
For a given `<feature>` (e.g., `document-type`):
1. **Types:** `src/types/<feature>.ts`
   - Define `<Feature>Schema` (Zod), `<Feature>Payload`, `<Feature>Dto`, and `Page<Feature>`.
2. **API:** `src/lib/<feature>-api.ts`
   - Implement `fetch`, `create`, `update`, `delete`, and `toggleStatus` (or similar).
   - Use `/api/proxy/v1/<feature>s` base path.
3. **Hooks:** `src/hooks/<feature>-hooks.tsx`
   - Implement `use<Feature>s`, `useCreate<Feature>`, `useUpdate<Feature>`, `useDelete<Feature>`, and `useToggle<Feature>Status`.
   - Implement orchestrator hook `use<Feature>Content` for DataTable state (pagination, sorting, search).
4. **UI:** `src/components/<feature>/`
   - `<feature>-content.tsx`: Main card with `DataTable`.
   - `<feature>-form-dialog.tsx`: Create and Edit dialogs using `builder/` components.
   - `<feature>-delete-dialog.tsx`: Delete confirmation.
5. **Page:** `src/app/(master)/master/<feature>/page.tsx`
   - Server component rendering the content component.

## Constraints
- **Language:** All UI labels, placeholders, and toast messages MUST be in **Bahasa Indonesia**.
- **Icons:** Use `@tabler/icons-react`.
- **Validation:** Ensure code passes `bun run lint` (Biome).
- **Integrity:** Update `src/app/(master)/master/template.tsx` to add the new tab.

## Example
`scaffold-master-data document-type`
